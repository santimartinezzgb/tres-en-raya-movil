## Migración a TypeScript (resumen)

- Qué: Proyecto convertido a TypeScript. Se añadieron `tsconfig.json`, `App.tsx` e `index.ts`.
- Por qué: Mejor tipado, autocompletado y detección temprana de errores.
- Cómo probar:

```bash
npm install
npm run type-check
npm start
```

Nota: `index.ts` reemplaza la entrada anterior `index.js`. Si quieres, elimino `index.js`.
